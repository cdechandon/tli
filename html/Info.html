Page d'information

Cette page présente les développements que nous avons réalisés, nos sources, les auteurs, ainsi que les ressources bibliographiques et la webographie que nous avons utilisé.

  • Développements:

    • Pour realiser ce TP nous avons du apprendre a dynamiser notre site WEB avec AJAX comme en TLI d'annee 3 mais aussi avec une nouvelle technologie: PHP. Le PHP s'execute cote serveur donc nous avons du utilise une technologie LAMPP pour pouvoir realiser notre site WEB. De plus nous avons eu a organiser notre site WEB, nous avons donc choisi d'utiliser une architecture MVC (Model Vue Controlleur) pour separie les entites du site WEB. Sur ce site WEB il a ete demande d'implementer une gestion d'utilisateur, donc apres une verification d'information faite par AJAX il fallait stocker les informations de l'utilisateur dans une base de donnee. Pour cela nous avons utilisé les bases MySQL et avons appris a utiliser PDO afin d'ameliorer la maintenance de notre site WEB si nous basculons de technologie pour les bases de donnee. Afin de garantir une certaine confidentalite nous avons choisis de stocker un mot de passe hacher de l'utilisateur. Nous avons mis en place un routeur pour que l'utilisateur ne puisse acceder au site que par un seul point d'entree. Cela garantira une certaine securite du site WEB. Pour ce qui concerne la vue de notre site WEB, nous avons utilisé SMARTY qui est un generateur de templates, cela permet de bien separer la vue du controlleur.
